## What is DevStart?

<img align="right" width="400" src="https://images.unsplash.com/photo-1555066931-4365d14bab8c?w=600&q=80" alt="Dark code editor with neon syntax highlighting" />

DevStart started as a **GoMyCode bootcamp CLI assignment** вҖ” document nine terminal commands, push to GitHub, done. It became something bigger.

It's now a full interactive learning platform covering the three pillars every developer needs: **the command line**, **Git**, and **HTML**. Everything вҖ” 8 pages of content, games, simulators, flashcards, quizzes, a live code editor, and a dictionary вҖ” ships as a **single `index.html` file** with no dependencies, no build step, no server.

**Why single-file?**
- Email it. They open it. It works. Fully offline.
- No `npm install`. No Webpack. No config files.
- Deploy anywhere вҖ” GitHub Pages, Netlify, a USB stick.

```
Clone вҶ’ Open вҶ’ Learn.
That's the whole process.
```

Built with the dark terminal aesthetic of the tools developers live in вҖ” black background, neon green accents, JetBrains Mono throughout.

## Contributing

```bash
# Fork on GitHub, then:
git clone https://github.com/YOUR_USERNAME/gomycode-cli-assignment.git
cd gomycode-cli-assignment

git checkout -b fix/your-description

# Edit index.html вҶ’ test in browser

git add .
git commit -m "fix: describe your change"
git push origin fix/your-description
# Open a pull request
```

**Commit conventions:**

| Prefix | Use for |
|--------|---------|
| `feat:` | New feature or section |
| `fix:` | Bug fix |
| `style:` | CSS / visual, no logic change |
| `refactor:` | Restructure, same behaviour |
| `docs:` | README or comments |
| `chore:` | Rename, housekeeping |
