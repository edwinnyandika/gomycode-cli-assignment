## What is DevStart?

DevStart started as a **GoMyCode bootcamp CLI assignment** вҖ” document nine terminal commands, push to GitHub, done. It became something bigger.

It's now a full interactive learning platform covering the three pillars every developer needs before anything else: **the command line, Git, and HTML**. Everything вҖ” 8 pages of content, games, simulators, flashcards, quizzes, a live code editor, and a dictionary вҖ” ships as a **single `index.html` file** with no dependencies, no build step, and no server required.

```
Open file вҶ’ Learn instantly.
```

Built with the dark terminal aesthetic of the tools developers actually use. Black background, neon green accents, JetBrains Mono throughout.

## Pages in Detail

### вҢЁпёҸ CLI Tasks
Nine filesystem operations fully documented with the exact commands run during the bootcamp assignment. Each task shows the command, what it does, and the expected output вҖ” tested on both **Git Bash (Windows)** and **Termux (Android)**.

A live **interactive terminal simulator** lets you type real CLI commands in the browser вҖ” `mkdir`, `cd`, `touch`, `ls`, `cat`, `rm`, `echo`, `pwd`, `mv`, `cp` вҖ” against a virtual in-memory filesystem. No terminal required.

### рҹҢҝ Git Commands
Every essential Git command organised into five categories: Setup, Tracking, Branching, Remote, and History. A complete feature-branch workflow walkthrough from `git init` to `git merge`, plus a **live Git simulator** that lets you run `git init`, `add`, `commit`, `branch`, `checkout`, and `log` in the browser.

### `</>` HTML Guide
HTML tag anatomy broken down visually, a semantic elements glossary, a common-tags reference table, and a full HTML5 boilerplate you can copy instantly. Covers everything a beginner needs to write real markup.

### рҹ’» Code Editor
A three-pane in-browser editor (HTML / CSS / JavaScript) with a **live rendered preview** that updates as you type. Ships with 7 starter snippets вҖ” hello world, flexbox layout, dark card, form, animation, grid, and a fetch API example. Supports VS Code keyboard shortcuts.

### рҹ§  Quiz
Twenty questions spanning CLI, Git, and HTML with:
- Answer selection via mouse **or** keyboard (`1` `2` `3` `4` + `Enter`)
- Progress dot track showing correct/wrong/current across all 20 questions
- **Web Audio API** sound feedback (high tone = correct, low tone = wrong)
- Score counter that updates live per answer
- **Canvas confetti** explosion when you score вүҘ 70%

### рҹғҸ Flashcards
Forty-plus cards across CLI, Git, and HTML with:
- **3D flip animation** with spring physics on reveal
- вҳҶ / вҳ… mastery toggle with persistent tracking across the session
- Mastery progress bar that fills as you mark cards learned
- Category filters with live count badges
- One-click download as a `.txt` file to study offline

### рҹ“– Lexicon
Fifteen curated developer terms, each with:
- **Plain English definition** вҖ” no jargon, just meaning
- **Technical depth definition** вҖ” precise developer framing
- Syntax-highlighted code example
- SVG visual (DOM tree, flexbox diagram, git commit timeline, branch graph, tag anatomy, attribute diagram, and more)

The search bar connects to the **Free Dictionary API** (`api.dictionaryapi.dev`) вҖ” free, open source, no API key needed. Search any word: if it's in the curated list, you get the full DevStart entry; if not, the live API returns real dictionary results with pronunciation audio, definitions, synonyms, and source links.

### рҹҺ® Code Game
Three game modes across three difficulty levels вҖ” **72 unique challenges** total:

| Mode | What you do | Levels |
|------|-------------|--------|
| рҹ§© **Code Fill-In** | Drag or click code pieces into blank spots in real syntax-highlighted snippets | 8 Г— Beginner, Pro, Advanced |
| рҹҺЁ **Visual Decode** | Read colour-coded code and identify what each highlighted part represents | 8 Г— Beginner, Pro, Advanced |
| рҹҗӣ **Bug Hunt** | One bug is hiding in a code block вҖ” click the broken line before time runs out | 8 Г— Beginner, Pro, Advanced |

**Difficulty stages:**
- `Beginner` вҖ” 45 seconds, clear prompts, basic syntax (doctype, mkdir, h1, git add)
- `Pro` вҖ” 30 seconds, multi-concept questions (flexbox, arrow functions, -b flag, selectors)
- `Advanced` вҖ” 20 seconds, real-world complexity (async/await, --amend, :first-child, nullish coalescing)

Every round features a live HUD (score, streak with рҹ”Ҙ, countdown timer, progress bar), sound feedback, an animated score ring on the results screen, automatic next-level suggestions when you score вүҘ 75%, and confetti for scores вүҘ 80%.

## Assignment Context

This project was built for the **GoMyCode Full-Stack Bootcamp** вҖ” CLI & Git fundamentals checkpoint. The original requirements:

- Use the terminal to complete 9 filesystem tasks
- Test on Git Bash (Windows) and Termux (Android)
- Document every command with its output
- Initialise a Git repo, commit the work, push to GitHub

The assignment was submitted as required. Then the file kept growing.

Each feature was added to make the documented material actually interactive вҖ” first a terminal simulator so you could practice the commands, then a Git simulator, then a quiz to test retention, then flashcards, then a full game with three modes and three difficulty levels. Every addition stayed in the same file.

## Design System

The platform runs a single dark terminal theme throughout:

```css
/* Colour tokens */
--bg:       #080c08   /* near-black with green tint вҖ” base background   */
--green:    #39ff6a   /* neon green вҖ” primary accent, active states      */
--green2:   #2bcc55   /* mid green вҖ” hover states                        */
--amber:    #ffc542   /* warm amber вҖ” attributes, warnings, streak       */
--blue:     #4dc9ff   /* electric blue вҖ” tags, info, links               */
--purple:   #bd93f9   /* soft purple вҖ” JS selectors, part-of-speech      */
--red:      #ff5555   /* terminal red вҖ” errors, wrong answers            */
--text:     #c8e6c9   /* light green-white вҖ” body text                   */
--text-dim: #5a7a5a   /* dimmed green вҖ” secondary text                   */
```

**Typography**
- `JetBrains Mono` вҖ” every piece of code, every UI label, every terminal line
- `Syne` вҖ” page titles, section headings, score numbers

**Motion principles**
- Glow pulses on active nav tabs
- Spring-physics 3D flip on flashcards
- Animated SVG ring fill on game results
- Canvas particle confetti (130 particles, physics-based)
- Countdown timer urgency animation below 8 seconds
